# Angular7 for CMS

### Project Technical Structure
***
*  angular7
*  ng-zorro-antd
*  angular/router
*  angular/http
*  rxjs
*  webpack4
*  less

### Install

***
project address: (`git clone`)

```
git clone git@github.com:xpioneer/cms-fe-angular7.git
```
install node_modules with `yarn`

```
yarn #in your command terminal
```
***
### Run

start server: (http://localhost:9100)

```
yarn run start
```

publish production

```
yarn run build
```
